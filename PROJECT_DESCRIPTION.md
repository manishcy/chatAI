# Project Description

This file provides a brief overview of the chatAI project.

chatAI is an AI-powered chat application designed to facilitate seamless conversations between users and various AI models. The project leverages modern web technologies and integrates multiple AI services to deliver an interactive and user-friendly experience.

Key Features:
- Real-time chat interface
- Model selection for different AI engines
- Responsive design
- Extensible architecture for adding new features

For more details, refer to the README.md file or explore the source code in the `src` directory.
